1. This is a Net implementation of Kontagent Client API, written in C#.
  2. This is totally self contained and does not rely on any other libraries. It has been tested in .Net 3.5.
  3. To use it, unzip the source code project. It is setup to build as a Class Library. Build it using Visual Studio and add it as a reference in your app.
  4. API Usage:
    • Init the KontagentRequestInstance in your your Global.asax Application_BeginRequest, by calling KontagentRequestInstance.Init. Pass it the Kontagent API and Secret keys.
    • If you use a Master Page for all your pages, place a call to KontagentRequestInstance.OnPageLoad in your Master Page's Page_Load event. Else place it each of your Page's Page_Load events that you want to track.
    • All URL's that you use for Invites, Feed Forms, etc. must be modified with Kontagent tracking parameters. Use the KontagentRequestInstance.MungeXXX functions to do that.
    • After invites are sent, call the KontagentRequestInstance.InviteSent function to track recipients.
    • Call KontagentRequestInstance.RecordUserInfo to pass user's demographic info to Kontagnet. To avoid calling it everytime, you can call it only once on install.
  5. To have requests sent over to Kontagent's Test Server, define KontagentTestMode key in your web.config AppSettings and set its value to 1. Remember to remove the key or set it to 0 to switch back to Live servers.
